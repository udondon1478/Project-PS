# レーティング判定フローチャート

このガイドは、商品に適切なレーティングを付与するためのインタラクティブなフローチャート形式の判定ツールです。各質問に「はい」「いいえ」で答えることで、適切なレーティングを判定できます。

## 使い方

1. 質問1から順番に回答してください
2. 各質問に対して「はい」または「いいえ」で答えてください
3. 回答に応じて次の質問に進んでください
4. 最終的なレーティングが判定されるまで続けてください

---

## 質問フロー

### 質問1: 性的な部位や行為のデータが含まれていますか?

3Dモデルとして性器が作られている、陰部のテクスチャがリアルに描かれている、または性行為のアニメーションデータが含まれていますか?

- **はい** → **レーティング: Explicit** ✓
- **いいえ** → [質問2](#質問2-乳首や性器周辺の描写がありますか)

---

### 質問2: 乳首や性器周辺の描写がありますか?

サムネイルや商品画像において、乳首、肛門、陰毛などが描写されていますか?（テクスチャの書き込み含む）また、それらが透けて見える衣装ですか?

- **はい** → **レーティング: Questionable** ✓
- **いいえ** → [質問3](#質問3-性的な道具や拘束具が含まれますか)

---

### 質問3: 性的な道具や拘束具が含まれますか?

バイブレーター、ディルドなどの性具、または拘束具などの性的な用途を主とするアイテムが含まれていますか?

- **はい** → **レーティング: Questionable** ✓
- **いいえ** → [質問4](#質問4-露出度の高い衣装ですか)

---

### 質問4: 露出度の高い衣装ですか?

下着、マイクロビキニ、または肌の露出が極端に多い衣装（ニプレスのみ等）ですか?

- **はい** → **レーティング: Sensitive** ✓
- **いいえ** → [質問5](#質問5-性的な部位を強調していますか)

---

### 質問5: 性的な部位を強調していますか?

サムネイルや画像で、胸や臀部、股間を強調するアングルや、扇情的なポーズ（M字開脚など）をとっていますか?

- **はい** → **レーティング: Sensitive** ✓
- **いいえ** → [質問6](#質問6-下着が見える構図ですか)

---

### 質問6: 下着が見える構図ですか?

パンチラやスカートの中を覗き込むようなアングル（ローアングル）の画像が含まれていますか?

- **はい** → **レーティング: Sensitive** ✓
- **いいえ** → [質問7](#質問7-全年齢向けとして問題ありませんか)

---

### 質問7: 全年齢向けとして問題ありませんか?

職場や学校などで閲覧しても問題ない、性的な要素を含まないコンテンツですか?

- **はい** → **レーティング: General** ✓
- **いいえ** → **レーティング: Sensitive** ✓

*迷った場合や、わずかでも配慮が必要な要素がある場合はSensitiveを選択してください*

---

## レーティング一覧

### General (全年齢)
- 完全に非性的
- 職場や公共の場で安全
- 不適切な要素なし

### Sensitive (軽微な配慮が必要)
- 軽度に性的、示唆的
- 露出度の高い服装
- パンチラなどのファンサービス
- 軽度な暴力や血液
- 自傷、違法薬物

### Questionable (要注意)
- ヌード(性器以外)
- 性的接触や行為の示唆
- 明確な性的小道具
- グラフィックな暴力

### Explicit (成人向け)
- 露骨な性器露出
- 明示的な性行為
- 体液の描写
- 極端なグラフィック暴力(NSFL)

## Web実装時の仕様

このフローチャートは、将来的にWebインターフェースとして実装される予定です。

### 想定されるUI要素

1. **質問カード**: 各質問を1つずつ表示
2. **はい/いいえボタン**: 大きく分かりやすいボタン
3. **進捗インジケーター**: 現在の質問番号と残り質問数
4. **戻るボタン**: 前の質問に戻る機能
5. **結果カード**: 最終的なレーティングを大きく表示
6. **詳細説明**: レーティングの理由と該当する基準を表示
