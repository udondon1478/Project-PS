# レーティング判定フローチャート

このガイドは、商品に適切なレーティングを付与するためのインタラクティブなフローチャート形式の判定ツールです。各質問に「はい」「いいえ」で答えることで、適切なレーティングを判定できます。

## 使い方

1. 質問1から順番に回答してください
2. 各質問に対して「はい」または「いいえ」で答えてください
3. 回答に応じて次の質問に進んでください
4. 最終的なレーティングが判定されるまで続けてください

---

## 質問フロー

### 質問1: 露骨な性器の露出がありますか?

性器(陰部、ペニス、肛門など)が露出している、または明確に見える描写はありますか?(検閲の有無を問わず)

- **はい** → **レーティング: Explicit** ✓
- **いいえ** → [質問2](#質問2-明示的な性行為の描写がありますか)

---

### 質問2: 明示的な性行為の描写がありますか?

性交、フェラチオ、手淫、指入れ、セックストイの使用などの明示的な性行為が描写されていますか?

- **はい** → **レーティング: Explicit** ✓
- **いいえ** → [質問3](#質問3-極端なグラフィック暴力がありますか)

---

### 質問3: 極端なグラフィック暴力がありますか?

ゴア、切断、内臓露出などの極端にグラフィックな暴力表現(NSFL相当)がありますか?

- **はい** → **レーティング: Explicit** ✓
- **いいえ** → [質問4](#質問4-性器以外のヌードがありますか)

---

### 質問4: 性器以外のヌードがありますか?

乳首、乳輪、素肌のお尻など、性器以外の性的部位の露出がありますか?

- **はい** → **レーティング: Questionable** ✓
- **いいえ** → [質問5](#質問5-露出度の高い服装や透ける服装がありますか)

---

### 質問5: 露出度の高い服装や透ける服装がありますか?

水着、ランジェリー、下着、透ける服、ウェット服などの露出度の高い服装がありますか?

- **はい** → [質問5-A](#質問5-a-性的部位が強調されていますか)
- **いいえ** → [質問6](#質問6-軽度な性的接触や示唆がありますか)

#### 質問5-A: 性的部位が強調されていますか?

胸、お尻、股間などの性的部位に焦点を当てた構図やポーズですか?

- **はい** → **レーティング: Sensitive** ✓
- **いいえ** → [質問5-B](#質問5-b-服越しに性器や乳首が透けていますか)

#### 質問5-B: 服越しに性器や乳首が透けていますか?

薄い服を通して乳首や性器の輪郭が明確に見えますか?

- **はい** → **レーティング: Questionable** ✓
- **いいえ** → **レーティング: Sensitive** ✓

---

### 質問6: 軽度な性的接触や示唆がありますか?

耳噛み、股間への触れ合い、胸への触れ合い、激しいキス、または性的示唆のあるポーズやジェスチャーがありますか?

- **はい** → [質問6-A](#質問6-a-性行為の直接的な示唆ですか)
- **いいえ** → [質問7](#質問7-パンチラやアップスカートなどがありますか)

#### 質問6-A: 性行為の直接的な示唆ですか?

フェラチオのジェスチャーや、性行為を直接示唆する行為がありますか?

- **はい** → **レーティング: Questionable** ✓
- **いいえ** → **レーティング: Sensitive** ✓

---

### 質問7: パンチラやアップスカートなどがありますか?

パンツが見える、スカートの下が見える、またはそれに類似するファンサービス的な表現がありますか?

- **はい** → **レーティング: Sensitive** ✓
- **いいえ** → [質問8](#質問8-性的な小道具や背景要素がありますか)

---

### 質問8: 性的な小道具や背景要素がありますか?

コンドーム、セックストイ、ラブホテルなど、性的な文脈を示唆する小道具や背景がありますか?

- **はい** → [質問8-A](#質問8-a-使用中または露骨に見えますか)
- **いいえ** → [質問9](#質問9-グラフィックな暴力表現がありますか)

#### 質問8-A: 使用中または露骨に見えますか?

性的な小道具が使用されている、または非常に明確に描写されていますか?

- **はい** → **レーティング: Questionable** ✓
- **いいえ** → **レーティング: Sensitive** ✓

---

### 質問9: グラフィックな暴力表現がありますか?

大量の血液、深刻な怪我、死の描写、虐待など、グラフィックまたは不穏な暴力表現がありますか?

- **はい** → **レーティング: Questionable** ✓
- **いいえ** → [質問10](#質問10-軽度な暴力や血液がありますか)

---

### 質問10: 軽度な暴力や血液がありますか?

軽い擦り傷、打撲、切り傷、少量の血液など、軽度な暴力表現がありますか?

- **はい** → **レーティング: Sensitive** ✓
- **いいえ** → [質問11](#質問11-自傷や違法薬物の描写がありますか)

---

### 質問11: 自傷や違法薬物の描写がありますか?

自殺や自傷の描写、または違法薬物の使用が含まれていますか?

- **はい** → **レーティング: Sensitive** ✓
- **いいえ** → [質問12](#質問12-完全に非性的で安全な内容ですか)

---

### 質問12: 完全に非性的で安全な内容ですか?

職場や公共の場で安全に閲覧できる、完全に非性的で不適切な要素のない内容ですか?

- **はい** → **レーティング: General** ✓
- **いいえ** → **レーティング: Sensitive** ✓

*迷った場合や、わずかでも配慮が必要な要素がある場合はSensitiveを選択してください*

---

## レーティング一覧

### General (全年齢)
- 完全に非性的
- 職場や公共の場で安全
- 不適切な要素なし

### Sensitive (軽微な配慮が必要)
- 軽度に性的、示唆的
- 露出度の高い服装
- パンチラなどのファンサービス
- 軽度な暴力や血液
- 自傷、違法薬物

### Questionable (要注意)
- ヌード(性器以外)
- 性的接触や行為の示唆
- 明確な性的小道具
- グラフィックな暴力

### Explicit (成人向け)
- 露骨な性器露出
- 明示的な性行為
- 体液の描写
- 極端なグラフィック暴力(NSFL)

## Web実装時の仕様

このフローチャートは、将来的にWebインターフェースとして実装される予定です。

### 想定されるUI要素

1. **質問カード**: 各質問を1つずつ表示
2. **はい/いいえボタン**: 大きく分かりやすいボタン
3. **進捗インジケーター**: 現在の質問番号と残り質問数
4. **戻るボタン**: 前の質問に戻る機能
5. **結果カード**: 最終的なレーティングを大きく表示
6. **詳細説明**: レーティングの理由と該当する基準を表示
