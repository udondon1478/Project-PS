# Delayed CodeRabbit Fix Skill

このスキルは、CodeRabbitのAIレビューが完了するのを待機してから、自動修正フローを開始するために使用します。PR作成直後など、レビューがまだ到着していないタイミングで有効です。

## 手順

### 1. 待機設定

ユーザーにPR番号を確認し、バックグラウンドではなく、**このセッションで**待機することを確認します。

### 2. 待機実行

以下のコマンドを使用して、15分間（900秒）待機します。
※ ユーザーの希望があれば時間は調整可能です。

```bash
echo "⏳ CodeRabbitのレビュー完了を待機しています（15分）..." && \
sleep 900 && \
echo "✅ 待機時間が終了しました。レビュー取得を開始します。"
```

### 3. 自動修正の開始

待機コマンドが正常に完了したら、直ちに `fix_coderabbit` スキルを実行に移してください。

> **Note**: 待機中にユーザーが別の操作を行いたい場合は、このスキルではなく `Task` ツールを使用してバックグラウンドで待機させることを検討してください。しかし、修正プロセス自体は対話的に行う必要があるため、基本的にはこのまま待機することを推奨します。
